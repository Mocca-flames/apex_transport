# Plan: Fix Mobile Navigation Toggle for Apex Transport Website

## Problem
- Mobile toggle button exists in HTML (`#mobileToggle`) but has no click handler
- Navigation menu (`#navLinks`) never opens/closes on mobile
- No JavaScript to toggle the `.active` class that controls visibility

## Solution
Add standard, accessible mobile navigation with these features:

### 1. Toggle Functionality
- Add click event listener to `#mobileToggle` button
- Toggle `.active` class on `#navLinks` to slide menu in/out
- Add hide/show on scroll (existing pattern: `.nav-hidden` class)

### 2. Best Practices Implementation
- Accessible: Update `aria-expanded` attribute on toggle button
- Close menu when nav links are clicked (single-page app UX)
- Prevent body scroll when menu is open (mobile UX)
- Handle escape key to close menu
- Use event delegation for performance

### 3. Location
Add script just before closing `</body>` tag (after all HTML content, before any other potential scripts)

### 4. Code Structure
```javascript
document.addEventListener('DOMContentLoaded', function() {
  const mobileToggle = document.getElementById('mobileToggle');
  const navLinks = document.getElementById('navLinks');
  const nav = document.querySelector('nav');

  // Toggle mobile menu
  mobileToggle.addEventListener('click', function() {
    const isOpen = navLinks.classList.toggle('active');
    this.setAttribute('aria-expanded', isOpen);
    // Prevent body scroll when menu open
    document.body.style.overflow = isOpen ? 'hidden' : '';
  });

  // Close menu when link clicked
  navLinks.querySelectorAll('a').forEach(link => {
    link.addEventListener('click', () => {
      navLinks.classList.remove('active');
      mobileToggle.setAttribute('aria-expanded', 'false');
      document.body.style.overflow = '';
    });
  });

  // Hide nav on scroll down, show on scroll up
  let lastScroll = 0;
  window.addEventListener('scroll', () => {
    const currentScroll = window.pageYOffset;
    if (currentScroll > 100) {
      if (currentScroll > lastScroll) {
        nav.classList.add('nav-hidden');
      } else {
        nav.classList.remove('nav-hidden');
      }
    }
    lastScroll = currentScroll;
  });

  // Close menu on escape key
  document.addEventListener('keydown', (e) => {
    if (e.key === 'Escape' && navLinks.classList.contains('active')) {
      navLinks.classList.remove('active');
      mobileToggle.setAttribute('aria-expanded', 'false');
      document.body.style.overflow = '';
    }
  });
});
```
