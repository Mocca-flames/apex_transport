# Plan: Africa Coverage Grid — Polished Visual Design with Status Emojis

## Goal
Redesign the coverage section to be visually polished, unconstrained, and use distinct status emojis alongside country flags. Mobile-first, scannable at a glance.

---

## 1. Current State Analysis

Existing coverage section issues:
- **Constrained layout**: `aspect-ratio: 1/1` on `.map-container` forces a square box that feels cramped
- **Text-heavy labels**: "HQ Hub", "Active", "Specialist" are plain text — not visually distinctive
- **No status emojis**: User wants map/location emojis to differentiate country types visually
- **4-column grid**: Too tight on mobile, cards feel squished
- **Missing visual breathing room**: Cards touch edges, no padding/margin around the grid

---

## 2. Proposed Solution

### A. Remove Constraints
- **Remove** `aspect-ratio: 1/1` from `.map-container`
- **Remove** `max-width: 700px` — let the grid breathe to full width
- **Remove** `overflow: hidden` and fixed borders that box it in
- **Add** generous padding around the grid (`48px` desktop, `24px` mobile)

### B. Status Emojis (replacing text labels)
Instead of text like "HQ Hub" / "Active" / "Specialist", use distinct emojis:

| Status | Emoji | Meaning |
|--------|-------|---------|
| Hub | 🏠 | Headquarters / Primary hub |
| Active | 🛣️ | Active corridor / regular route |
| Specialist | ⚠️ | Specialist routing / complex clearance |

These emojis appear:
- **On the card** (below the country flag, large and colorful)
- **In the popup** (when tapped)
- **In the legend** (replacing colored dots)

### C. Card Design Refresh
- **Larger cards**: More padding (`40px 24px` desktop, `32px 16px` mobile)
- **Bigger flag emojis**: `56px` desktop, `40px` mobile
- **Status emoji**: `24px`, positioned as a badge/icon
- **Remove bottom border** (`::before` colored bar) — replace with **subtle left border** or **top accent**
- **Hover effect**: Slight lift (`translateY(-4px)`) + shadow instead of just background change
- **Active press**: Scale down (`0.97`) for tactile feedback

### D. Grid Layout
- **Desktop**: 4 columns, `24px` gap, full width
- **Tablet**: 2 columns
- **Mobile**: 2 columns, `16px` gap, edge-to-edge with padding

### E. Legend Refresh
- Replace colored dots with **emoji + text**:
  - 🏠 Hub
  - 🛣️ Active
  - ⚠️ Specialist
- Larger text, more spacing

## 3. Implementation Steps

### Step 1: Remove Constraints (CSS)
```css
.map-container {
    position: relative;
    width: 100%;
    /* REMOVE: aspect-ratio: 1/1; */
    /* REMOVE: max-width: 700px; */
    margin: 0 auto 48px;
    /* REMOVE: border: 1px solid var(--color-border); */
    /* REMOVE: background: #f8f9fa; */
    /* REMOVE: overflow: hidden; */
    padding: 48px;
}
```

### Step 2: Update Country Cards (HTML)
Replace `.country-role` text with status emoji:
```html
<div class="country-item hub" role="listitem" tabindex="0" data-country="za">
    <span class="country-flag">🇿🇦</span>
    <span class="country-name">South Africa</span>
    <span class="country-status">🏠</span>
</div>
```

### Step 3: Redesign Card CSS
```css
.countries-grid {
    display: grid;
    grid-template-columns: repeat(4, 1fr);
    gap: 24px; /* Increased from 1px */
    margin-bottom: 48px;
}

.country-item {
    background: #ffffff;
    padding: 40px 24px;
    text-align: center;
    cursor: pointer;
    transition: all 0.3s ease;
    border-radius: 8px; /* Added */
    border: 1px solid var(--color-border);
    /* REMOVE: position: relative; */
    /* REMOVE: ::before colored bar */
}

.country-item:hover {
    transform: translateY(-4px);
    box-shadow: var(--shadow-md);
    border-color: var(--color-primary);
}

.country-item:active {
    transform: scale(0.97);
}

/* Status-specific border colors */
.country-item.hub { border-top: 4px solid var(--color-primary); }
.country-item.active { border-top: 4px solid var(--color-secondary); }
.country-item.specialist { border-top: 4px solid #eab308; }

.country-flag {
    font-size: 56px;
    margin-bottom: 16px;
}

.country-name {
    font-size: 20px;
    margin-bottom: 8px;
}

.country-status {
    font-size: 24px;
    display: block;
}
```

### Step 4: Update Legend
```html
<div class="map-legend">
    <span class="legend-item"><span class="status-emoji">🏠</span> Hub</span>
```

## 4. Visual Checklist

- [ ] No square/aspect-ratio constraint on container
- [ ] Cards have rounded corners and subtle border
- [ ] Status emojis (🏠 🛣️ ⚠️) clearly visible on each card
- [ ] Hover lifts card with shadow
- [ ] Active press scales down
- [ ] Color-coded top border (orange/blue/yellow)
- [ ] Generous padding around grid
- [ ] Mobile: 2 columns, comfortable tap targets

---

## 5. Summary

Remove the constrained square container, replace text status labels with emojis (🏠 🛣️ ⚠️), add rounded corners and hover lift effects, and let the grid breathe with full width and generous spacing. Mobile gets 2 columns with larger touch targets.
