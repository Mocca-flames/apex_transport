# Plan: Remove Mobile Scroll-Jacking from Services Section

## Summary
Disable the scroll-driven (scroll-jacking) behavior in the Services section **on mobile only** (≤1024px). Desktop remains unchanged. Panels stack vertically in normal document flow on mobile while preserving visual design.

## How Scroll-Jacking Works Currently
- `js/services.js` `initServices()` sets up a scroll listener + IntersectionObserver
- `.services-track` height is `calc(6 * 100svh)` — creates 600svh of scroll space
- `.services-sticky` is `position: fixed` — pins to viewport while you scroll through the tall track
- JS computes scroll progress and cross-fades between 6 panels stacked in a CSS grid

## Changes

### 1. `js/services.js` — Skip scroll-jacking on mobile
- At the top of `initServices()`, after getting the track element, check `window.matchMedia('(max-width: 1024px)').matches`
- If mobile:
  - Add class `services--static` to the track
  - Inject mobile images into each panel (keep existing image injection logic)
  - Set all panels to `is-active` once they're in the DOM
  - `return` early — skip scroll listeners, IntersectionObserver, progress dots, keyboard handlers
- Desktop path: no changes, existing logic runs as-is

### 2. `css/components.css` — Static mobile mode CSS
Add new rules inside the existing `@media (max-width: 1024px)` block (after the current services rules):

```css
/* Static mode: no scroll-jacking, panels stack vertically */
.services--static .services-track,
.services-track.services--static {
  height: auto;
}

.services-track.services--static .services-sticky {
  position: static;
  opacity: 1 !important;
  pointer-events: auto !important;
  height: auto;
  display: block;
}

.services-track.services--static .services-panels {
  display: flex;
  flex-direction: column;
  gap: var(--space-16);
  padding: var(--space-8) var(--container-pad);
}

.services-track.services--static .services-panel {
  opacity: 1 !important;
  transform: none !important;
  pointer-events: auto !important;
  position: relative;
  grid-area: auto;
  grid-row: auto;
  grid-column: auto;
}

.services-track.services--static .services-progress {
  display: none;
}
```

## Files Modified
- `js/services.js`
- `css/components.css`

## Design Preservation
- Panel structure (counter, title, image, body, button) stays identical
- Mobile image injection still runs
- Fonts, colors, spacing — all stay the same
- No visual changes to desktop at any viewport size
